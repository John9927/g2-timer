Tutorial

How it works

A quick guide to control the timer from the glasses, including start, pause, reset, layout editing, and the meaning of each gesture.

Tap Open, confirm, start, pause, resume, or change the selected layout value.
Double tap Go back, close a prompt, cancel a reset confirmation, or exit from Home.
Swipe Change preset shortcuts, move across action choices, or move between layout fields.
App only If the glasses layout label is set to App only, layout changes stay available only on the phone.
Tutorial

Main flows

Start Start a timer

Open Timer on the glasses, swipe to pick a preset, then tap. If layout editing is enabled on the glasses, tap opens a small menu where you choose Start or Layout. If layout editing is disabled, tap starts the timer immediately.

Pause Pause and resume

While the timer is running, tap to open the action menu. Swipe to keep Pause selected and tap to confirm. When paused, tap again to open actions and confirm Resume.

Reset Reset safely

From the running or paused action menu, swipe to Reset timer and tap. A confirmation screen appears. Tap again to reset, or double tap to cancel.

Layout Change layout on the glasses

If allowed by the phone setting, open Timer, tap, choose Layout, then swipe between fields and tap to change each value. Double tap returns to the timer screen.

Tutorial

Gesture map

Home Enter the timer

From the glasses home screen, tap to open Timer. Double tap exits the app.

Idle Choose the preset

Swipe cycles through the preset shortcuts. Tap opens the next action. Double tap goes back to the glasses home screen.

Running Open timer actions

Tap opens the action menu. Swipe switches between available actions. Double tap closes the action menu or returns to the dashboard, depending on the current screen.

Done After completion

When the timer reaches zero, tap resets it back to the selected preset so you can start again quickly.

Tip: if you want the cleanest possible glasses UI, set Glasses layout label to App only in phone settings. In that mode the layout stays editable only from the app.
